# github-api-search
Project was built using GitHub API.  An AJAX call to API with username, password and userhandle pulls all the repositories and  commit comments.  All data is displayed in DOM using handlebarsjs. 

## Tech Stack
 * Javascript
 * JQuery
 * AJAX
 * HTML5
 * Handlebars
 * CSS3
